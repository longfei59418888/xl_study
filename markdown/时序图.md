### 介绍

- 展示各个参与者之间是如何交互的
- 结构元素
    - 参与者
    - 消息
    - 激活框
    - 注释
    - 循环(loop)
    - 选择(alt)
    - 可选(opt)
    - 并行(Par)
- 连线和箭头
    - -> 实线无箭头
    - ->> 实线有箭头
    - --> 虚线无箭头
    - -->> 虚线有箭头
- 激活框
    - 开始：activate <对象>
    - 结束：deactivate <对象>
- 注释
    - note left of <对象> : 在对象左边
    - note right of <对象> : 在对象右边
    - note over <对象1>,<对象2> : 在对象1和对象2上面
- 选择(alt)

```dotenv
alt 条件说明
	[消息流]
else 条件说明
	[消息流]
else 条件说明
	[消息流]
end
```

- 可选(opt)

```dotenv

opt 条件说明
[消息流]
end

```
