
#### 网络层总结
- IP协议：在首部添加目的网络号和源网络号
- 主机号/MAC：每一个 MAC 地址和 主机号 都一一映射
- 路由器设备：每个接入路由器的设备都会发出 地址解析协议请求 
- 地址协议请求：告诉适配器保存的 MAC 地址和主机号
- 路由器中只会解析 IP 首部的 目的地址，找到对应局域网
- 局域网通过 IP / MAC 地址映射找到对应主机







#### 传输层
- 两个主要协议：TCP和UDP
- TCP：在传输数据时候会建立稳定连接，3次握手、四次挥手
- UDP：不会建立稳定连接，传输直接传输数据报，主机接收也不会有确认返回
